# Wire Harness（线束）简介

1. 如字面上的意思：把一条线或多条线束在一起成为一个连接线组，就是 Wire Harness 通常使用于内部配线，也就是机器（设备）中或是机壳（箱）内的线组

2. 组成部分：电线，端子，塑件。

3. 应用：电子产品、电脑、影音系统、家电产品、自动控制系统、汽车、公共运输系统、交通设备、机械设备、监控系统、航太工业、医疗设备等…

## 一、裁线

1. 使用裁切机设定好参数，把线材裁切成所需长度。

![裁线机操作界面图片]

## 二、剥外皮

1. 根据剥皮长度调整好脱皮机的定位螺杆。
2. 将线材剥皮端顶住定位杆，紧靠剥皮刀口不可放歪。
3. 脚踏气动开关。
4. 重复以上动作。

# 三、剪編織

1. 先將編織挑开。
2. 再用剪刀沿外被口根部剪去全部編織。

# 五、套地线热缩套管

1. 将地线拧紧，保证不叉开。
2. 选定需求规格套管套入地线根部。
3. 地线末端弯折，避免套管掉出。
4. 批量周转至烘套管机处烘紧。

# 六、烘地线热缩套管

1. 将热缩套管按图面要求位置定位。
2. 放到热风机上方约离口5～10CM处烘紧。

# 十一. 測試

1. 當班使用的測試機必需由品管確認過參數等 OK 後方可使用。
2. 成品與對應的母座要配套使用，接插時要平插平拔。
3. 按要求設定測試條件參數：
4. 測試時上下 45 度擺動三次測試
5. 觀查顯示屏確認結果，并將良品與不良品進行對應處理。

絕緣阻抗：5MΩ MIN 300V/DC.
導體阻抗：1Ω MAX.

**PASS** 表示通過，
若顯示 **FAIL**，表示不良。

# 十二、烘热缩套管

1. 将套管按图面或 SOP 尺寸定位。
2. 固定尺寸后将产品移到热风机上方约离出风口 5～10CM 处将套管烘紧。

![Heat shrink tubing installation process showing measurement, positioning over heat gun, and final result - includes 4 instructional photos demonstrating the steps]

# 十三、外观尺寸全检

1. 要在 10 倍的放大镜下作业。
2. 按 SOP 要求作业。
3. 对不良品及良品进行标识区分。

主要检验：
1. 成品表面是否有沾污。
2. 成品线体是否有划伤及损坏。
3. 线位是否正确及插入到位。
4. 尺寸在公差要求内。

# 十四. 包装

1. 按 SOP 要求将成品定为规定数理一把，要求首尾对应整齐。
2. 分别用所配的橡皮筋及气泡袋扎一束。
3. 将对应的纸箱对折后封底。
4. 在封底后的纸箱内加入防水袋。
5. 按顺序依次装入箱中，并贴上对应的标签。
6. 外箱标签要注明料号，生产日期，重量。
